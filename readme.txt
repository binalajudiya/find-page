== Description ==

Find a page details from slug on one click. It will show ID, name and Edit link.

= Key Features in Find page Editions =

* **Page:** Find a page ID, Name or edit link from the slug

== Installation ==

1. Upload the `find-page` folder to the directory `/wp-content/plugins/`.
2. Activate the plugin using the 'Plugins' menu in WordPress.

== Frequently asked questions ==

## How to use

1. First Activate Plugin.
2. Go Select to Find Page settings Menu from Settings Tab and savings settings. 
3. Go to page -> Find page and enter slug to get Page details
